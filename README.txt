To compile and run Weather Client Application :
-----------------------------------------------

1. Clean classes:
   
   ant clean

   That command will delete the classes folder
   Which contains compiled file from the client
   programm


2. Compile the Weather client:

   ant compile 

   The command will referenced the stubs folder
   which is output/ and compile without any need
   to passing any arguments

3. Run the Weather Client:

   ant run

   The command will run the application
